Append Storage is de-facto event storage for Synqra. It is designed to be simple, efficient, and scalable. It provides a way to store events in a sequential manner, allowing for easy retrieval and processing.

It should not be bound to events only, and for this both Model Type and Keys - are generic. This allows for a wide range of use cases, such as storing snapshots, projections, or any other type of data that can be represented as a stream.
